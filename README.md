# Stuhi HQ Visitor Counter Display
This is an add-on to the [HQ Visitor Counter](https://github.com/Novelty-Labs-Stuhi/visitorcounter-mini-project). This fetches the current amount of people at the office via Wi-Fi and displays it on an LED matrix. When the value changes, a cool animation is played depending on if someone left, or just arrived.
